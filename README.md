# Guardio Cybersecurity Awareness Chatbot

## 📌 Project Overview

Guardio is a Cybersecurity Awareness Chatbot developed using **C#** and **Windows Presentation Foundation (WPF)**.

The chatbot educates users about important cybersecurity topics such as:
- Password Safety
- Phishing
- Online Scams
- Privacy
- Malware
- VPN Security
- Firewall Protection

The application includes:
- A professional WPF graphical user interface
- Voice greeting
- Username validation
- Keyword recognition
- Random chatbot responses
- Sentiment detection
- Memory recall
- Conversation flow handling

---

# 🎯 Features

## ✅ GUI Interface
- Professional WPF graphical interface
- Modern colours and styling
- Multiple grids/screens
- Interactive chatbot layout

## ✅ Voice Greeting
- WAV voice greeting plays when the application launches

## ✅ Username Validation
The application validates usernames by:
- Preventing empty usernames
- Preventing numeric usernames
- Checking minimum username length

## ✅ Keyword Recognition
The chatbot recognises keywords such as:
- password
- phishing
- scam
- privacy
- malware
- vpn
- firewall
- hacker

## ✅ Random Responses
The chatbot uses dictionaries and lists to generate random responses for a more natural conversation experience.

## ✅ Sentiment Detection
The chatbot detects emotions such as:
- worried
- frustrated
- confused
- nervous
- curious

## ✅ Memory Recall
The chatbot stores:
- user name
- favourite cybersecurity topic
- last discussed topic

and recalls them later to personalise the interaction.

## ✅ Conversation Flow
Supports follow-up prompts such as:
- "Tell me more"
- "Another tip"
- "Explain more"

# 🧠 How the Chatbot Works

## Keyword Recognition

The chatbot scans user input and identifies cybersecurity keywords to generate relevant responses.

## Sentiment Detection
